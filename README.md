AmethystKQ
==========

A low-resource optimized Paper 1.21.11 fork for 20-player servers.

**Links:**
- Forum: [faq.amethystcore.com](https://faq.amethystcore.com)
- Discord: [amethystcore.com/discord](https://amethystcore.com/discord)

- Website: [amethystcore.com](https://amethystcore.com)

How To (Server Admins)
------
AmethystKQ is distributed as a paperclip jar. Download from [kq.amethystcore.com](https://kq.amethystcore.com) and run it like a normal Minecraft server jar.

```
java -jar AmethystKQ-1.21.11.jar --nogui
```

How To (Compiling From Source)
------
Requires JDK 21 and an internet connection.

```bash
./gradlew applyPatches
./gradlew createReobfPaperclipJar
```

The compiled jar will be in `paper-server/build/libs/`.
